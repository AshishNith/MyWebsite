Learn the Basics of CSS

CSS stands for Cascading Style Sheets. It is used to style HTML documents.

Basic Example

                
body {
    background-color: lightblue;
}

h1 {
    color: white;
    text-align: center;
}

p {
    font-family: verdana;
    font-size: 20px;
}

Use the above CSS code to style your HTML elements.
